# 🛒 Promociones Argentina - Base de Datos Actualizada Diariamente

[![Actualización Diaria](https://github.com/paprikacc/promos-argentina/actions/workflows/update-promos.yml/badge.svg)](https://github.com/paprikacc/promos-argentina/actions)

Base de datos completa con promociones bancarias en supermercados de Argentina, actualizada automáticamente todos los días.

## 🏪 Comercios Incluidos

- **Carrefour** - Descuentos bancarios
- **Jumbo** - Promociones del día
- **Día** - Medios de pago
- **Coto Digital** - Promociones
- **Makro** - Beneficios bancarios
- **MODO** - Promos multi-comercio

## 💳 Métodos de Pago

- Visa
- Mastercard
- American Express
- MODO
- Mercado Pago
- Cuenta DNI
- Y más...
